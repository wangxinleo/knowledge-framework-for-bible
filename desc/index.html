          # 👉指引

          《圣经脑图》是一个致力于帮助圣经读者建立圣经知识体系的辅助方案。

          通过思维导图的模式，串联每一章书卷的知识零散点，以树状图的结构配合大脑思考，完成圣经阅读。

          本项目于2020年12月15日在阅读圣经时突然诞生，诞生之后就立即撰写相关编写方案及团队招募策略。
